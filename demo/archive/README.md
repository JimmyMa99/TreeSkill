## 当前推荐

请优先使用：

```bash
python -m treeskill
python -m treeskill sealqa-aso
```

对应主线是：
- `Kode` 负责前向
- `ASO` 负责 skill/program 修改
- `SealQA lifecycle` 负责展示 `root -> generate -> evolve -> prune -> merge`
